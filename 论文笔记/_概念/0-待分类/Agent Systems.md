# Agent Systems

## 定义

基于 Foundation Models 构建的智能体系统，通过工具调用、记忆和规划解决复杂任务。

## 核心要点

1. **FM为核心**: 以 Foundation Models 作为推理引擎
2. **工具扩展**: 通过 bash、文件操作等工具扩展能力
3. **记忆机制**: 可集成短期/长期记忆支持复杂任务
4. **自主决策**: 可自主调用工具和规划执行步骤
