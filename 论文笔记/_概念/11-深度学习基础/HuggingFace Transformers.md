# HuggingFace Transformers

## 定义

HuggingFace Transformers 是最流行的开源 Transformer 模型库，提供预训练模型、训练和推理工具。

## 核心要点

1. **模型库**: GPT、BERT、LLaMA 等主流模型
2. **易用性**: 高级 API 简化模型使用
3. **推理效率**: 基础实现，未针对 serving 优化

## 代表工作

- [[vLLM]] 支持 HuggingFace 模型格式
