# Godel Machine

## 定义

Schmidhuber (2003) 提出的自改进AI理论模型，通过可证明有益的自我修改实现递归改进。

## 核心要点

1. **可证明改进**: 只有数学上可证明有益的修改才被执行
2. **完全自引用**: 理论上可修改任何部分包括验证机制
3. **最优性保证**: 在有限时间内趋向最优策略
4. **实用性限制**: 实际实现困难，验证成本高
