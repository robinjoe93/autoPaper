# Deformable Object Manipulation

## 定义

可变形物体操作是指机器人对形状可变的物体（如布料、绳子、软组织等）进行的抓取、移动、折叠、穿线等操作任务。

## 核心难点

可变形物体具有以下特点，使其操作难度远超刚体：

1. **高维配置空间**: 物体状态由大量节点/粒子位置描述，而非单一刚体姿态
2. **无固定参考帧**: 物体形状随操作动态变化，不存在稳定的坐标系
3. **复杂的物理动力学**: 需要精确建模弹性、塑性、摩擦等物理特性
4. **视觉感知困难**: 物体变形导致外观多变，难以建立稳定的视觉特征

## 应用场景

- 家庭服务机器人（毛巾折叠、衣物处理）
- 医疗机器人（手术缝合、软组织操作）
- 工业自动化（柔性材料加工）
- 辅助机器人（毛绒玩具、被子整理）
