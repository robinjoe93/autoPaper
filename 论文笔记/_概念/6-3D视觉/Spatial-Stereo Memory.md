# Spatial-Stereo Memory (SSM / SSM++)

## 定义

Spatial-Stereo Memory (SSM) 是 [[WorldStereo]] 系列模型中用于维护局部细节一致性的记忆机制。它借鉴传统立体匹配和参考修复的思想，通过检索参考视图并与目标视图空间拼接来恢复精细细节。

## 核心思想

传统全注意力方法处理离散、无序的检索帧时学习困难。SSM 通过：
1. 离散检索相关参考视图
2. 空间拼接每个检索视图与其对应目标视图
3. 限制注意力接受域到拼接对

有效恢复细节并建立拼接对的对应关系。

## SSM++ 改进

[[WorldStereo 2.0]] 中的 SSM++ 包含以下改进：

### 1. 主分支集成
- 直接将检索关键帧纳入主 DiT 分支（不再使用单独记忆分支）

### 2. RoPE 修改
- 目标帧与检索参考视图水平拼接（宽度 $2W$）
- 每个检索视图继承其配对目标帧的时间索引

### 3. 全注意力策略
- 移除注意力接受域限制（除 cross-attention）
- 通过全自注意力学习全局上下文

### 4. 隐式相机嵌入
- 使用 7 维向量（四元数 + 平移）替代显式点图引导
- 通过 3 层 MLP 编码为相机 token
- 通过零初始化添加到特征

## 检索策略

### 训练阶段
- **时间错位检索**: 用于现有多视图数据，随机选择 30%-90% 时间重叠的帧
- **多轨迹检索**: 用于合成数据，基于 3D FoV 相似性从替代轨迹选择

### 推理阶段
- 全景细分视图作为初始记忆库
- 增量更新生成的关键帧
- 最多检索 $T_r$ 个关键帧（$T_r < T_{kf}$）

## 与 GGM 配合

| 机制 | 作用 | 特点 |
|------|------|------|
| [[GGM]] | 全局几何一致性 | 粗略结构、点云渲染 |
| [[SSM++]] | 局部细节一致性 | 精细细节、视图检索 |

## 优势

1. **选择性检索**: 只检索最相关关键帧，减少冗余计算
2. **灵活引导**: 隐式相机嵌入替代显式点图
3. **全局学习**: 全注意力策略学习全局上下文
4. **数据增强**: 运动模糊、颜色抖动、裁剪增强鲁棒性
